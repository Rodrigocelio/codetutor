Hello World em C++

Tutorial passo a passo para VS Code (com compilador MinGW-w64)

1 Instalar o Compilador (MinGW-w64)

  • Baixe em winlibs.com ou MSYS2
  • Instale em caminho sem espaços: C:\mingw64
  • Selecione: x86_64 + posix + seh

2 Adicionar ao PATH

Adicione C:\mingw64\bin às variáveis de ambiente.

3 Verificar Instalação 

g++ --version

Se aparecer a versão, está tudo certo! ✅

4 Criar Projeto no VS Code

  1. Crie uma pasta: C:\projetos\hello
  2. Abra no VS Code → File > Open Folder
  3. Crie main.cpp

5 Escrever o Código 

#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}

6 Compilar e Executar 

g++ main.cpp -o hello
./hello

💡 Dica: Instale a extensão C/C++ no VS Code para melhor experiência.
